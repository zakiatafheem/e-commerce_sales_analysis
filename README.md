# E-commerce_sales_analysis

## Project Overview

This project focuses on analyzing ecommerce sales data using Power BI.
The dataset was cleaned and transformed directly within Power BI, followed by building an interactive dashboard to extract meaningful business insights.

## Data Cleaning (Power BI)
The raw Excel dataset was preprocessed using Power BI Power Query:

Removed missing and duplicate values,
Corrected data types (dates, numbers, categories),
Handled inconsistent entries,
Created new calculated columns where required,
Renamed columns for better readability.

## Dashboard Features
The dashboard provides interactive visualizations to understand sales performance:

-> Total Sales, Profit, and Orders KPIs
-> Monthly Sales Trends
-> Category-wise Sales Analysis
-> Region-wise Performance
-> Customer insights (if applicable)
-> Filters/Slicers for dynamic analysis
